August 17, 2021

Music Genre Classification

I started this project out of interest in music analytics, and the data sets can be found on Kaggle. Here is a picture of the data set that contains 22 features and 26752 samples. Data Set For my analysis I decided to focus on 9 features, which includes popularity, danceability, energy, acousticness, instrumentalness, loudness, liveness, speechiness, and valence. Most of these features are described on Spotify’s audio analysis page, but all the variables range from 0 to 1 and are based on Spotify’s algorithm.
March 20, 2020

Human Activity Recognition Project

This project is about using data collected from a Samsung smartphone to classify whether the human was sitting, standing, laying, walking, or walking upstairs/downstairs. For our features we used the 3-axial linear acceleration and angular velocity collected by the accelerometer and gyroscope, which are built in the phone. These features were sampled at 50Hz or 50 measurements per second. Since the raw data is quite noisy, the raw data has been filtered and extracted to 561 features that characterize the measurement in time and frequency.
February 15, 2020

Cushing Syndrome: Discriminant analysis and Baye's Classifier

One of the main topics in machine learning is classification, so this project was about comparing Linear Discriminant Analysis(LDA), Quadratic discriminant analysis(QDA), and the optimal Baye’s Classifier. The first plot shows three samples of multivariate Gaussian distributions, the LDA boundary(magenta), and the Baye’s classifier boundary(black). The second part of the project was to predict the diagnosis of Cushing Syndrome using concentrations of steroid metabolites, specifically tetrahydrocortisone and pregnanetriol.
